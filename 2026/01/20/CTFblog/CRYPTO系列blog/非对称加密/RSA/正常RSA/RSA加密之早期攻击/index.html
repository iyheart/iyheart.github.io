RSA加密之早期攻击

发表于2026-01-20|更新于2026-01-22|CTFCRYPTORSA
|字数总计:2.2k|阅读时长:9分钟|阅读量:
  • 对于目前的RSA加密来说，只介绍一些正常的RSA加密题型，而不会详细归纳RSA加密中考察某个数论知识点的题目。（当然如果是leak类型的题目还是会归纳的，但是如果是单单考察一个求ϕ(n)\phi(n)ϕ(n)的公式，那我觉得没什么必要）
  • 接下来就介绍一下针对RSA加密的早期攻击方式，以便后续的RSA加密的学习。接下来就先来了解一下RSA加密的早期攻击：对于RSA加密早期的攻击，本篇博客中主要学习的是共模攻击、循环攻击、共明文攻击，对于相关明文攻击个人认为篇幅比较多，所以单开一篇博客进行论文的细读，进行学习。

共模攻击

  • 共模攻击英文全称为Common Modulus Attack，出现该攻击的原因是由于一个早期提出的密码学协议共模协议，该协议的具体内容为：

    • 由一个密钥机构（即可信第三方）生成一个RSA模数，并向系统内的用户分发使用同一模数的和法密钥对。
    • 在该协议中，用户的私钥仅为(d,N)(d,N)(d,N)，因此用户并不知道模数NNN的因数分解即不知道(p,q)(p,q)(p,q)的值。
    • 该方案的初衷是只有中央密钥机构掌握该公共模数的分解信息。

  • 在1983年，Simmons在这篇文章A "weak" privacy protocol using the RSA crypto algorithm. 指出，当同一明文使用具有相同模数且公钥指数互素的两个不同公钥进行加密时，会存在协议失效的问题。接下来具体说明一下：

    • 给定两个密文c1,c2c_1,c_2c1​,c2​，以及两个公钥对(e1,n)，(e2,n)(e_1,n)，(e_2,n)(e1​,n)，(e2​,n)，其中公钥对中(e1,e2)=1(e_1,e_2)=1(e1​,e2​)=1。Simmons证明了在这种已知条件下，可以很容易地计算出原始明文。
    • 由于(e1,e2)(e_1,e_2)(e1​,e2​)互素，那么我们就可以轻易地求出整数a1a_1a1​和a2a_2a2​，a1e1+a2e2=1a_1e_1+a_2e_2=1a1​e1​+a2​e2​=1（可以由裴蜀定理得到该式子）
    • 对于任意明文mmm，给定的c1=me1 mod( N),c2=me2 mod( N)c_1=m^{e_1}~mod(~N),c_2=m^{e_2}~mod(~N)c1​=me1​ mod( N),c2​=me2​ mod( N)，只需要计算c1a1∗c2a2 mod( N)c_1^{a_1}*c_2^{a_2}~mod(~N)c1a1​​∗c2a2​​ mod( N)，即可得到明文。
    • 因为在ZNZ_NZN​中有：c1a1∗c2a2=ma1e1∗ma2e2=ma1e1+a2e2=m mod( N)c_{1}^{a_1}*c_{2}^{a_2}=m^{a_1e_1}*m^{a_2e_2}=m^{a_1e_1+a_2e_2}=m~mod(~N)c1a1​​∗c2a2​​=ma1​e1​∗ma2​e2​=ma1​e1​+a2​e2​=m mod( N)
    • 这类攻击可以由任何能够获取公钥并且观察到这两个密文的人实施

  • 在1984年，DeLaurentis在这篇文章A further weakness in the common modulus protocolfor the RSA cryptoalgorithm证明了共模协议是完全不安全的。他指出只要掌握任意一组公钥—私钥对，就足以计算出在相同模数下对应与任何其他公钥的有效私钥，具体定理如下：

    • 设(e,N)(e,N)(e,N)是一个有效的RSA公钥，其私钥对应的是(d,N)(d,N)(d,N)；设(e1,N)(e_1,N)(e1​,N)是另一个有效的公钥，并且满足e1≠ee_1≠ee1​​=e。

    • 在已知(e,d,N,e1)(e,d,N,e_1)(e,d,N,e1​)的情况下，公钥(e1,N)(e_1,N)(e1​,N)对应有效的解密指数d1d_1d1​能在关于log(N)log(N)log(N)的多项式时间内计算得到。

    • d1d_1d1​的计算公式如下：d1=e1−1 mod( ed−1gcd(e1,ed−1))d_1=e_1^{-1}~mod(~\frac{ed-1}{gcd(e_1,ed-1)})d1​=e1−1​ mod( gcd(e1​,ed−1)ed−1​)

    • 证明如下：

      • 由于ϕ(N)\phi(N)ϕ(N)是λ(N)\lambda(N)λ(N)的倍数，所以密钥等式就可以写成ed−1=kλ(N)ed-1=k\lambda(N)ed−1=kλ(N)，kkk是一个正整数。
      • 对于有效的公钥指数e1e_1e1​，必须满足gcd(e1,λ(N))=1gcd(e_1,\lambda(N))=1gcd(e1​,λ(N))=1，因此就有gcd(e1,kλ(N))=gcd(e1,k)=k′gcd(e_1,k\lambda(N))=gcd(e_1,k)=k'gcd(e1​,kλ(N))=gcd(e1​,k)=k′，也就有k′∣kk'|kk′∣k。
      • 设k~=kk′≥1\tilde{k}=\frac{k}{k'}≥1k~=k′k​≥1，所以就有ed−1gcd(e1,ed−1)=kλ(N)k′=k~λ(N)\frac{ed-1}{gcd(e_1,ed-1)}=\frac{k\lambda(N)}{k'}=\tilde{k}\lambda(N)gcd(e1​,ed−1)ed−1​=k′kλ(N)​=k~λ(N)。
      • 因此对于私钥指数d1d_1d1​就满足e1d1=1+k1(k~λ(N))e_1d_1=1+k_1(\tilde{k}\lambda(N))e1​d1​=1+k1​(k~λ(N))，其中k1k_1k1​是正整数。
      • 因此就有e1d1≡1 ( mod λ(N))e_1d_1\equiv1~(~mod~\lambda(N))e1​d1​≡1 ( mod λ(N))，所以d1d_1d1​是公钥(e1,N)(e_1,N)(e1​,N)的一个有效私钥指数。
      • 所以可以在关于log(N)log(N)log(N)的多项式时间内计算得到d1d_1d1​

    • 此外DeLaurentis利用一个归因与Simmons的思想证明了：在已知单个公钥-私钥对的情况下，可以使用概率多项式时间的Las Vegas算法对模数进行分解（Las Vegas算法是结果一定正确，只是运行时间是随机的随机算法）。在给定eee和ddd的条件下，只需要计算ϕ(N)\phi(N)ϕ(N)的一个倍数，即ed−1=kϕ(N)ed-1=k\phi(N)ed−1=kϕ(N)，然后应用Miller的结论（该结论就是在已知ϕ(N)\phi(N)ϕ(N)的某个倍数时，可以以概率方式分解NNN），因此共模协议不安全。

广播攻击

  • 另一个出现协议失效的情况就是几个相关的明文消息被小公钥指数和不同的模数加密。同样的，对这个失效的协议攻击经常指的是Håstad's Broadcast attack，也就是Håstad广播攻击。该攻击分为两类，一类就是共明文攻击，另一类就是消息相关攻击。
  • 这里主要介绍共明文攻击，该协议失效是出现在相同的明文mmm被几个公钥(e,Ni)(e,N_i)(e,Ni​)加密，每个公钥都有相同的公钥指数eee和不同的模数NiN_iNi​。这个协议失效的攻击在1985年由Håstad发表的两篇文章On using RSA with low exponent in a public key network和Solving simultaneous modular equations of low degree
  • 该攻击的具体定理如下：
    • 设(e,N1),...,(e,Nl),l≥e(e,N_1),...,(e,N_l),l≥e(e,N1​),...,(e,Nl​),l≥e，这些公钥对都是有效的RSA公钥，其中模数Ni,i=1,...,lN_i,i=1,...,lNi​,i=1,...,l两两互素
    • 设N0=min{N1,...,Nl}N_0=min\{N_1,...,N_l\}N0​=min{N1​,...,Nl​}，N=Πi=1lNiN=\Pi^{l}_{i=1}N_iN=Πi=1l​Ni​
    • 对于任何一个消息m<N0m<N_0m<N0​，给定ci=me mod ( Ni)c_i=m^e~mod~(~N_i)ci​=me mod ( Ni​)和(e,Ni),i=1,...,l(e,N_i),i=1,...,l(e,Ni​),i=1,...,l，明文mmm是能在关于log(N)log(N)log(N)的多项式时间内计算出来的。
    • 证明如下：
      • 由于模数N1,...,NlN_1,...,N_lN1​,...,Nl​两两互素，由中国剩余定理可以用cic_ici​和NiN_iNi​得到C≡me mod( N)C\equiv m^e~mod(~N)C≡me mod( N)。
      • 因为m<N0m<N_0m<N0​，这就使得me<N1N2...Nl=Nm^e<N_1N_2...N_l=Nme<N1​N2​...Nl​=N，因此C=meC=m^eC=me。
      • 在整数范围内计算C=meC=m^eC=me的e次方根，即可得到明文mmm，由于所有计算都可以在关于log(N)log(N)log(N)的多项式时间内完成，因此结论成立。
